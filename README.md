# yape

Este proyecto se basa en un proyecto peruano de Laboratoria, está estructurado en base a los siguientes requerimientos: 
- Consumo de API
- Diseño similar a la aplicación original
- Validaciones de data
- uso de Gulp
- uso de Sass
- Buenas prácticas
